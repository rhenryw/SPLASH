SPLASH - **S**ecure **P**roxy for **L**ive **A**udiovisual **SH**ell

SPLASH is a Command Line-like proxy that is command based for a more unique access style.

Just open deployment on a desktop (or download and run `python3 -m http.server {port}`)

Base commands:
`help` - See list of commands (just use this I'm not going to type it all out)
